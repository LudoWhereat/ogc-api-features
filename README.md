# Setup

## Install  NodeJS
Go to https://nodejs.org/en/ and install the LTS

Test if the install was OK on the command prompt:

```
node
```

If all was installed correctly, you will see:

```
Welcome to Node.js v12.18.1.
Type ".help" for more information.
>
```
